What is a relation?

A relation between two sets is a subset of their Cartesian product. It formalises the notion that elements from one set are "related to" elements of another (or the same) set.

Formal Definition
Binary Relation

A binary relation R from set A to set B is a subset of the Cartesian product A × B. Each element of R is an ordered pair (a, b) where a ∈ A and b ∈ B.

R ⊆ A × B  ⟺  R = { (a, b) | a ∈ A, b ∈ B, a R b }

When A = B, we call it a relation on A (a homogeneous / endorelation).

Key terminology

Cartesian Product

A × B = { (a, b) | a ∈ A, b ∈ B }. If |A| = m and |B| = n, then |A × B| = mn. A relation R ⊆ A × B.

A = {1,2}, B = {x,y}
A×B = {(1,x),(1,y),(2,x),(2,y)}
Domain & Range

The domain of R is the set of all first elements. The range (codomain) is the set of all second elements.

Dom(R) = { a | (a,b) ∈ R }
Ran(R) = { b | (a,b) ∈ R }
Identity Relation

The identity relation I_A on A contains exactly the pairs (a, a) for every a in A. Every element relates only to itself.

I_A = { (a, a) | a ∈ A }
Universal Relation

The universal relation U on A is A × A — every element relates to every other element including itself.

U = A × A (all pairs)

Representing relations

Ordered Pairs

List the pairs explicitly:
R = {(1,1),(1,2),(2,3)}

Boolean Matrix

M[i][j] = 1 if (aᵢ, aⱼ) ∈ R, else 0. Rows = domain, columns = range.

Digraph

Directed graph: an arrow from a to b means (a,b) ∈ R. Self-loops represent reflexive pairs.

Operations on relations

Inverse / Converse

R⁻¹ swaps all pairs: if (a,b) ∈ R then (b,a) ∈ R⁻¹.

R⁻¹ = { (b, a) | (a, b) ∈ R }
Composition

R∘S: (a,c) ∈ R∘S if ∃b such that (a,b)∈R and (b,c)∈S.

R∘S = { (a,c) | ∃b: (a,b)∈R ∧ (b,c)∈S }
Closure

The smallest superrelation of R satisfying a given property (reflexive, symmetric, or transitive closure).

Complement

R̄ contains all pairs in A×A that are NOT in R.

R̄ = (A × A) \ R

Properties of relations

Relations on a set A ⊆ A × A can have structural properties. These properties determine how elements are related and what kind of structure the relation induces.

Reflexive
∀a ∈ A: (a,a) ∈ R

Every element must relate to itself. The diagonal of the boolean matrix is all 1s.

Irreflexive (Strict)
∀a ∈ A: (a,a) ∉ R

No element relates to itself. Diagonal is all 0s. Mutually exclusive with reflexive (unless A = ∅).

Symmetric
∀a,b: (a,b)∈R → (b,a)∈R

If a relates to b, then b relates to a. The boolean matrix equals its own transpose: M = Mᵀ.

Antisymmetric
∀a,b: (a,b)∈R ∧ (b,a)∈R → a=b

If a relates to b and b relates to a, they must be the same element. Symmetric ≠ not antisymmetric.

Asymmetric
∀a,b: (a,b)∈R → (b,a)∉R

Implies both irreflexive and antisymmetric. Strictly stronger than antisymmetric.

Transitive
∀a,b,c: (a,b)∈R ∧ (b,c)∈R → (a,c)∈R

If a→b and b→c then a→c must hold. Missing transitivity pairs are a common analyser finding.

Total (Connex)
∀a,b ∈ A: (a,b)∈R ∨ (b,a)∈R

Any two elements are comparable — at least one direction of the pair must exist.

Trichotomous
∀a≠b: exactly one of (a,b)∈R, (b,a)∈R, a=b

Strict total order of comparability. Exactly one of the three conditions holds for any distinct pair.

Vacuous truth

Important edge case

Many properties are universally quantified with a hypothesis. When the hypothesis is never satisfied, the property holds vacuously. The empty relation on any set is vacuously symmetric, transitive, and antisymmetric — because there are no pairs to violate those conditions.

Special named types

Well-order
Total Order + Well-Founded

A total order where every non-empty subset has a least element. Example: ℕ under ≤. Enables transfinite induction.

Dense Order
Strict Total + Density

For any a < b there exists c with a < c < b. Example: ℚ under <. No "adjacent" elements exist.

Relations & mappings

A function (mapping) is a special kind of relation with the uniqueness constraint: each domain element relates to exactly one codomain element.

▶ Function (Map / Mapping) f: A → B

A function f: A → B is a relation where every a ∈ A appears as a first element exactly once.

∀a ∈ A, ∃! b ∈ B : (a,b) ∈ f

This is the uniqueness + totality condition. A relation is a function iff each row of its matrix has exactly one 1.

▶ Composition of Relations

Given relations R ⊆ A×B and S ⊆ B×C, their composition S∘R ⊆ A×C contains (a,c) whenever there exists b ∈ B with (a,b)∈R and (b,c)∈S.

S∘R = { (a,c) | ∃b ∈ B : (a,b)∈R ∧ (b,c)∈S }

Composition is associative: (T∘S)∘R = T∘(S∘R). For a relation R on A, powers R² = R∘R, R³ = R∘R∘R, etc. Transitivity holds iff R² ⊆ R (boolean matrix: M² ≤ M entrywise).

▶ Injective (One-to-One)

f is injective if distinct inputs map to distinct outputs — no two elements share an image.

f(a) = f(b) → a = b  ⟺  a ≠ b → f(a) ≠ f(b)

Necessary for a function to have a left inverse. Each column of the function matrix has at most one 1.

▶ Surjective (Onto)

f is surjective if every element in B is the image of at least one element in A. Range = Codomain.

∀b ∈ B, ∃a ∈ A : f(a) = b

Necessary for a right inverse. Each column has at least one 1.

▶ Bijective (One-to-One & Onto) isomorphism

A bijection is both injective and surjective. It establishes a perfect pairing between A and B.

f bijective ↔ ∃f⁻¹: B → A (two-sided inverse)

Bijections between finite sets require |A| = |B|. They define cardinality equality for infinite sets.

▶ Partial Function

A partial function may leave some elements of A undefined. Each domain element maps to at most one codomain element.

∀a ∈ A, ∃ at most one b ∈ B : (a,b) ∈ f

Common in computing: functions that may fail or be undefined (division, lookup, etc.).

▶ Equivalence Classes & Quotient Sets

Given an equivalence relation ~ on A, the equivalence class of a is:

[a] = { x ∈ A | x ~ a }

The set of all equivalence classes {[a] | a ∈ A} forms a partition of A, called the quotient set A/~. The natural projection π: A → A/~ mapping a ↦ [a] is surjective.

A = ⋃ [a],   [a] ∩ [b] = ∅ or [a] = [b]
▶ Hasse Diagrams (Partial Orders)

A Hasse diagram visualises a poset by drawing a line upward from a to b when a < b and there is no c with a < c < b (no element "between" them). Reflexive and implied transitive pairs are omitted — only the covering relation is drawn.

a ≺ b (a covered by b) ↔ a < b ∧ ¬∃c: a < c < b
